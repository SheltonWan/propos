## 一、测试说明

### 1.1 切换角色操作步骤

1. 打开 `http://localhost:3000`
2. 点击右下角底部导航最右侧"我的"→ 进入 Profile 页
3. 找到角色切换卡片，点击目标角色名称
4. 页面自动刷新为该角色视图，右上角显示当前角色姓名

### 1.2 角色缩写表

| 缩写 | 角色标识 | 模拟用户 | 职务 |
|------|---------|---------|------|
| **SA** | `super_admin` | 张总 | 超级管理员 |
| **OM** | `operations_manager` | 李经理 | 资产运营部总监 |
| **LS** | `leasing_specialist` | 王专员 | 租务专员 |
| **FS** | `finance_staff` | 赵会计 | 财务主管 |
| **MS** | `maintenance_staff` | 陈师傅 | 维修技工 |
| **PI** | `property_inspector` | 周楼管 | 楼管巡检员 |
| **RV** | `report_viewer` | 钱投资 | 投资人（只读） |

### 1.3 结果记录规则

- **通过** → 在"结果"列填写 `✅`
- **失败** → 在"结果"列填写 `❌`，并在备注列注明实际现象
- **跳过** → 填写 `—`，并注明原因
