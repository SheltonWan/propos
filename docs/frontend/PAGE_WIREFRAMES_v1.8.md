**交互流程**:
1. 输入邮箱 + 密码 → 点击「登录」
2. 按钮显示 loading 动画
3. 成功 → Admin: `router.push('/dashboard')` / uni-app: `uni.switchTab('/pages/dashboard/index')`
4. 失败 → 底部显示对应错误码消息
5. 二房东首次登录 → 强制跳转修改密码

| 语义 | Admin (Element Plus) | uni-app (wot-design) | 示例场景 |

| 中性/停用 | `type="info"` ⚪ | `type="default"` ⚪ | 非可租/已作废/已停用 |
| 进行中 | `type="primary"` 🔵 | `type="primary"` 🔵 | 执行中/处理中/草稿 |
