### 2.5 重置密码页 — Admin PC（仅 Web）

**Admin 路由**: `/reset-password?token=<token>` · **组件**: `ResetPasswordView.vue`

说明：
- 点击邮件中的重置链接后自动打开此页
- 从 URL query 中读取 `token`，若缺失则跳回登录页并提示"链接无效"
- Flutter / uni-app 不需要此页面（移动端用户通过浏览器打开链接后跳转到 Admin Web）

```
│   │ ********                  👁  │ │
│   └────────────────────────────────┘ │
│   ▓▓▓▓▓▓░░░░ 强度: 中               │

│   确认新密码                          │

│   │    正在跳转到登录页…             │ │
│   └────────────────────────────────┘ │
│                                      │
│   ▓▓▓ 错误状态 ▓▓▓                    │
│   ┌────────────────────────────────┐ │
│   │ ⚠ 重置链接已失效或已使用        │ │
│   │   [返回登录]                    │ │
│   └────────────────────────────────┘ │
│                                      │
└──────────────────────────────────────┘
```

**交互流程**:
1. 组件 mounted 时检查 URL `?token=` 是否存在，缺失则立即 `router.replace('/login')`
2. 输入新密码 + 确认密码 → 客户端校验一致性 + 复杂度
3. 提交 `POST /api/auth/reset-password {token, new_password}`
4. 成功：显示"已重置"，2s 后 `router.push('/login')`
5. `RESET_TOKEN_INVALID` / `RESET_TOKEN_EXPIRED`：显示错误状态，提供「返回登录」跳转

| 中性/停用 | `type="info"` ⚪ | `type="default"` ⚪ | 非可租/已作废/已停用 |
| 进行中 | `type="primary"` 🔵 | `type="primary"` 🔵 | 执行中/处理中/草稿 |
