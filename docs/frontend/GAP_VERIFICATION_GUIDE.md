## 模块 1：资产与空间

### Gap-M1-01 ✅ 改造记录新增/编辑表单

| 步骤 | 操作 |
|------|------|
| 1 | 进入**资产** → 选任意楼栋 → 选任意楼层 → 点击任意单元 |
| 2 | 向下滚动到"改造记录"区域 |
| 3 | 点击"**新增改造记录**"按钮 |

**预期结果**：底部弹出表单，包含改造类型下拉、日期选择、施工造价输入、备注、提交按钮。

---

### Gap-M1-02 ✅ 图纸版本管理 UI

| 步骤 | 操作 |
|------|------|
| 1 | 进入**资产** → 选任意楼栋 → 进入楼层平面图页 |
| 2 | 点击工具栏右上角**时钟/历史图标**按钮 |
| 3 | 在右侧抽屉中点击"**上传新版本**"按钮 |

**预期结果**：
- 右侧抽屉展开，显示版本时间线（版本号 / 上传时间 / 操作人 / 版本说明）
- 点击"上传新版本"后弹出上传对话框，含文件选择和版本说明输入

---

### Gap-M1-03 ✅ 单元拆分/合并工作流

| 步骤 | 操作 |
|------|------|
| 1 | 进入任意单元详情页 |
| 2 | 点击右上角**"⋯"更多操作**按钮 |
| 3 | 分别点击"**拆分单元**"和"**合并单元**" |

**预期结果**：
- 下拉菜单出现"拆分单元"（橙色剪刀图标）和"合并单元"（紫色图标）
- **拆分**：3步向导弹窗（① 选择拆分方式 → ② 填写新单元信息 → ③ 确认并提交）
- **合并**：相邻单元选择器 → 确认卡片 → 提交

## 模块 2：合同管理

### Gap-M2-01 ✅ 续签涨跌幅对比

| 步骤 | 操作 |
|------|------|
| 1 | 进入**合同**列表 → 点击任意合同卡片 |
| 2 | 在合同详情页点击"**发起续签**"按钮 |
| 3 | 在弹窗中修改"新月租金"字段 |

**预期结果**：弹窗内出现蓝色信息对比卡片，实时显示：

- 原合同末期月租
- 新合同起始月租
- 涨跌幅百分比（正值绿色 ↑，负值红色 ↓，零值蓝色）

---

### Gap-M2-02 ✅ 合同模板库管理

| 步骤 | 操作 |
|------|------|
| 1 | 进入**合同**列表页 |
| 2 | 点击右上角**"模板库"**按钮（书籍图标，灰色胶囊） |
| 3 | 切换写字楼 / 商铺 / 公寓标签 |
| 4 | 点击右上角 **+** 新增模板 |
| 5 | 点击模板卡片上的**编辑**图标 |
| 6 | 点击模板卡片上的**删除**图标 |

**预期结果**：
- 页面跳转至模板管理视图，顶部有三个业态标签页（各显示模板数量）
- 每张模板卡片含：模板名称、简短描述、标签（Tag 图标）、条款要点列表、编辑/删除按钮
- 点击 + / 编辑 → 底部弹出表单（模板名称 / 描述 / 标签 / 条款要点 + 保存按钮）
- 点击删除 → 居中确认弹窗，需二次确认

|------|------|

- 右侧抽屉展开，显示历史批次列表（批次号 / 数据类型 / 时间 / 操作人 / 成功/失败数）
