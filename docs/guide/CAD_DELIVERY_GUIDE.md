# CAD 图纸交付指南

> **版本**：v1.0 · 2026-04-28  
> **适用对象**：需要向 PropOS 系统上传楼层平面图的乙方设计院、物业测绘方

---

## 一、总体要求

PropOS 接受 **DXF 格式**（Drawing Exchange Format）的楼层平面图。DXF 是 Autodesk 发布的开放交换格式，AutoCAD、天正、中望 CAD、浩辰 CAD 等主流 CAD 软件均支持导出。

| 项目 | 要求 |
|------|------|
| 文件格式 | `.dxf`（不接受 `.dwg`） |
| 文件粒度 | 每栋楼一个 DXF 文件（含该栋所有楼层） |
| 单文件上限 | 50 MB |
| Model Space | 每个楼层平面图须包含「**X 层平面图**」标题文字，例如「1 层平面图」「地下 1 层平面图」 |
| 编码 | DXF R2010 及以上版本（AutoCAD 2010 格式），推荐 UTF-8 编码 |

---

## 二、DXF 导出步骤

### 2.1 AutoCAD

1. 打开 AutoCAD，在命令行输入 `SAVEAS`（或菜单：**文件 → 另存为**）
2. 在「文件类型」下拉框中选择 **AutoCAD 2010 DXF (\*.dxf)**
3. 文件名建议以楼栋命名，如 `A栋.dxf`
4. 点击**保存**

> 若文件包含外部参照（Xref），请先执行 `XBIND` 绑定所有 Xref，再导出 DXF，确保平面图完整。

### 2.2 天正建筑（T-Arch）

1. 在天正菜单中选择**文件布图 → 图形导出**
2. 格式选择 **AutoCAD DXF**，版本选 **2010**
3. 勾选「导出 Model Space 所有实体」
4. 点击**确定**，选择保存路径

> 天正专有实体（TCH 块）在导出为标准 DXF 时会自动分解，接收端不需要安装天正插件。

### 2.3 中望 CAD（ZWCAD）

1. 菜单：**文件 → 另存为**（或快捷键 `Ctrl+Shift+S`）
2. 文件类型选择 **AutoCAD 2010 DXF (\*.dxf)**
3. 输入文件名，点击**保存**

### 2.4 浩辰 CAD（GstarCAD）

1. 菜单：**文件 → 另存为**
2. 文件格式选择 **DXF**，版本建议 **R2010**
3. 点击**保存**

---

## 三、楼层标识命名约定

系统根据 DXF 图纸内的**标题文字**自动识别楼层并匹配数据库记录。请确保每个楼层平面图在 Model Space 内包含符合以下规则的标题文字：

| 标题示例 | 系统解析结果 |
|---------|------------|
| `1 层平面图` / `F1 平面图` | 第 1 层 |
| `11 层平面图` / `F11 平面图` | 第 11 层 |
| `地下 1 层平面图` / `B1 平面图` | 地下 1 层 |
| `屋顶平面图` / `屋顶层平面图` | 屋顶层（按名称匹配） |
| `夹层平面图` / `M层平面图` | 夹层（按名称匹配） |

**注意**：标题文字须在图框内以独立文字对象存在（非块属性），字号建议不小于 300mm（1:100 出图比例下 3mm 印刷高度）。

## 四、不支持的情况

以下情况系统**无法处理**，请勿上传：

| 情况 | 原因 | 建议处理方式 |
|------|------|------------|
| 扫描件 PDF / JPG 平面图 | 不含矢量几何，无法切分 | 需使用 CAD 软件重新绘制或描绘为 DXF |
| 加密 DXF | 脚本无法读取内容 | 移除加密保护后再导出 |
| DWG 格式 | PropOS 不集成 DWG 解析器 | 在 CAD 软件中另存为 DXF |
| 无标题文字 | 系统无法识别楼层归属 | 在图纸中补充「X 层平面图」标题文字 |
| 单文件 > 50 MB | 超出上传限制 | 检查是否包含多余附着图像（`IMAGEDEF`），清理后重新导出 |
| 外部参照未绑定 | 导出 DXF 后参照体不完整 | 在 AutoCAD 中执行 `XBIND` → `XREF DETACH` 后重新导出 |

---

## 五、上传流程

1. 登录 PropOS 管理后台
2. 进入 **资产管理 → 楼栋详情**，点击页头「**上传整栋图纸（DXF）**」按钮
3. 在弹窗中拖拽或点击选择 `.dxf` 文件（单文件，≤ 50 MB）
4. 点击「**上传并切分**」，系统后台自动按楼层切分（通常耗时 1–2 分钟）
5. 切分完成后：
   - **已自动匹配**的楼层：直接在楼层列表中生效
   - **未匹配**的楼层：在弹窗中从下拉列表手动指派目标楼层
6. 关闭弹窗后，可在各楼层「楼层平面图」页查看导入的 SVG 图纸

## 六、常见问题

**Q：上传后提示「文件不包含有效的 DXF 结构」？**  
A：说明文件内容不是标准 DXF 格式。常见原因：上传了 DWG 文件（请在 CAD 软件中重新另存为 DXF），或文件在传输过程中损坏。

**Q：切分后所有楼层都在「未匹配」列表？**  
A：系统无法从文件名识别楼层，请手动在弹窗中逐一指派。建议下次导出 DXF 时，在每个楼层图框内补充「X 层平面图」标题文字。

**Q：某些楼层切分出的 SVG 显示为空白？**  
A：该楼层平面图可能位于 Paper Space 而非 Model Space，或所有图元在被冻结的图层上。请检查 CAD 图纸的图层状态，将所有需要导出的实体移至 Model Space 并解冻图层。

**Q：可以只替换某一层的图纸吗？**  
A：可以。进入 **楼层详情 → 楼层平面图**，点击右上角「**上传单层图纸**」，支持上传 `.svg` 或 `.dxf` 文件进行单层补丁替换。
