# PropOS 项目启动资料清单

> **文档版本**: v1.0  
> **更新日期**: 2026-04-17  
> **适用项目**: PropOS（Property Operating System）智慧物业资产运营管理平台  
> **使用说明**: 请按以下优先级分批提供，标注"★必须/立项前"的资料是项目开工的硬性前提，其余资料可在对应模块开发前补齐。

## 一、资产基础台账（必须·立项前）

| 资料名称 | 格式要求 | 字段要求 | 必填 | 提供责任人 |
|---------|---------|---------|------|----------|
| 写字楼单元清单 | Excel | 楼栋名、楼层、单元编号、套内面积、建筑面积、朝向、层高、装修状态、工位数、分隔间数、当前出租状态、参考市场租金（元/m²/月） | ★ | 运营团队 |
| 商铺单元清单 | Excel | 楼栋名、楼层、单元编号、套内面积、门面宽度、是否临街、层高、装修状态、当前出租状态、参考市场租金 | ★ | 运营团队 |
| 公寓单元清单 | Excel | 楼栋名、楼层、单元编号、套内面积、卧室数、独立卫生间数（是/否）、装修状态、当前出租状态、参考市场租金 | ★ | 运营团队 |

## 九、业务规则口径确认（必须·需求确认阶段）

以下规则请客户以书面形式 **明确确认**，避免后期反复返工。

| 规则类别 | 需确认的问题 | 默认值（系统内置，可调整） |
|---------|------------|--------------------------|
| **合同计费** | 首期/末期不足月，是否按自然日折算？ | 默认按自然日折算 |
| **合同计费** | 写字楼/商铺按建面还是套内计费？ | 默认按合同约定计费面积 |
| **税费口径** | 一般纳税人（增值税9%）还是简易征收（5%）？各租户适用税率是否相同？ | 每份合同单独标注 |
| **押金规则** | 押金是否允许转移至续签合同？冲抵违约金前是否需要审批？ | 允许转移；冲抵需财务确认 |
| **收款核销** | 多账单时，核销顺序是"先到期先核销"还是允许自由选择？ | 默认先到期先核销，允许手工调整 |
| **逾期判断** | 账期截止后第几天开始计逾期？ | 截止日后第1天 |
| **预警节点** | 到期预警：提前90/60/30天是否都要发？还是只选部分节点？ | 三个节点全部发送 |
| **预警接收人** | 每类预警的具体接收人是哪些岗位/人员？ | 见第2.3节 |
| **KPI 申诉窗口** | 快照冻结后允许申诉的天数？ | 7个自然日 |
| **NOI 口径** | 水电代收部分是否计入 EGI？物业管理费是否作为代收科目单独处理？ | 以客户实际操作确认为准 |
| **CapEx 阈值** | 单次维修超过多少元才列为资本性支出（CapEx，不进NOI）？ | 默认单次 > 5,000元且延长资产使用寿命 |
| **水电计价** | 是否有阶梯水价/阶梯电价？各业态的水价/电价单价是多少？ | 客户提供具体单价 |
| **商铺分成** | 保底租金+营业额分成的分成比例各家商铺相同还是不同？ | 每份合同单独配置 |
| **二房东审核期** | 二房东提交子租赁数据后，审核SLA是几个工作日？ | 2个工作日 |
